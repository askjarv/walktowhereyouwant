How to Create a Custom Theme

Follow these steps to create your own custom theme:

  1. Download the theme template by clicking the button below
  2. Edit the template file with your desired settings:
    • Set your theme name
    • Configure global theme colors (background, menu, text)
    • Add your milestone images and text
  3. Save your theme file
  4. Upload your theme file using the Theme Settings
